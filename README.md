# 🚀 Kuickpay HRMS

A modern Human Resource Management System (HRMS) built to streamline employee management, attendance tracking, payroll, recruitment, and organizational workflows through an intuitive and responsive web interface.

![Kuickpay HRMS Dashboard](./public/dashboard-preview.png)

> Simplifying HR operations with a fast, modern, and scalable solution.

---

## ✨ Features

### 👥 Employee Management
- Add, edit and delete employees
- Employee profiles
- Department assignment
- Employee status management

### 🏢 Department Management
- Create departments
- Manage department heads
- Department-wise employee listing

### 🕒 Attendance Management
- Daily attendance records
- Check-in & Check-out tracking
- Attendance reports
- Present/Absent statistics

### 🏖 Leave Management
- Apply for leave
- Leave approval workflow
- Pending leave requests
- Leave history

### 💰 Payroll
- Salary management
- Payroll processing
- Payroll history
- Pending payroll overview

### 🎯 Recruitment
- Candidate management
- Recruitment tracking
- Hiring workflow

### 📊 Reports & Analytics
- Employee statistics
- Attendance insights
- Payroll summaries
- HR analytics dashboard

### ⚙ Settings
- Profile management
- System configurations
- User management

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/kuickpay-hrms.git
```

Navigate into the project

```bash
cd kuickpay-hrms
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
